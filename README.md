This project is created as part of the Frontend, Backend, and Database Knowledge Test for The Tann Mann Foundation.

The application allows users to submit their Name, Phone Number, and Email Address through a simple frontend interface, which is then stored in a PostgreSQL database** using a backend REST API.

---

 Tech Stack
 Frontend
- React JS
- JavaScript
- HTML
- CSS

 Backend
- Node.js
- Express.js
- REST API
- PostgreSQL (`pg` library)

 Database
 PostgreSQL (Local setup)



 Features

- Simple Good Morning webpage
- Form with:
  - Name
  - Phone Number
  - Email Address
- Data submission from frontend to backend
- Backend API to store data in PostgreSQL
- Timestamp stored automatically in database
